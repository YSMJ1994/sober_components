# CollapseBox 下拉折叠容器
自动根据内容高度、可控的伸缩折叠容器

## 何时使用
下拉菜单，折叠面板等

## API
通过设置`show`属性来控制容器的显示隐藏，通过`animated`控制是否启动动画效果，通过`children`设置容器内容

传参说明如下：

|属性|说明|类型|默认值|
|---|---|---|---|---|
|show|显示状态|**boolean**|`false`|
|animated|动画效果|**boolean**|`true`|
|children|容器内容|**React Node**|无|
